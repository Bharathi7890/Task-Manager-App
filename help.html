Dashboard All Tasks About Help

📖 How to Use

  1. ➕ Add a new task on the Dashboard (title required, optional date/priority).
  2. 📋 Tasks appear in Today’s Tasks preview and All Tasks page.
  3. ☑️ Check the box to mark tasks as completed.
  4. ⏳ Unmarked tasks at day end are automatically set as not completed.
  5. 📊 Progress bar updates automatically.

💡 Tips

  • 🔎 Use filters to view Pending or Completed tasks.
  • 🗂 Tasks without deadlines remain until completed or deleted.
  • 🚫 Do not use Incognito mode; tasks won’t save there.
